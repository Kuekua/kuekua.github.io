分水岭算法的原理及实现

2017-07-08 • 分类: Image Processing • 标签: Algorithm, Image Processing

算法步骤：
1.构建图像梯度图像。
2.通过一定规则生成n个最初的注水区域（先验知识或局部梯度最小值）。
3.往注水区域内加水，当两注水区域即将合并时，记录下此时的边界。
4.当图像边缘彻底被分割成n个独立区域是算法结束。
